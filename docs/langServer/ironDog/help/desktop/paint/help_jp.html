ペイントゲームルール

  • ペイントは、5リール3列のビデオスロットで、フリースピンとボーナスゲームがあります。
  • スピンごとに固定の20ライン(ラインごとに1コイン)でプレーし、ベット合計は、ベットボタンに表示されます。
  • ベットの値を変えるには、ベットパネルを開き、矢印で変更します。
  • 最大ベットは、最大のベット値を読み込みます。
  • コイン/キャッシュは、残高の表示の切り替えをし、ペイアウトの値をキャッシュからコインで表示します。ベットの値は常にキャッシュで表示されます。コインの値: ベットの値/ライン数。
  • スピンボタンは、現在のベットの値でプレーします(設定で有効にしてある場合、スペースバーを押してプレーできます)。
  • 勝ちの組み合わせと配当は、配当表により決まります。
  • コインライン勝利は、配当表で表示された値と同じになります。
  • 有効なベットラインで左から右に並んだ、最も大きな勝ちの組み合わせだけが支払われます。
  • ワイルドシンボルは、すべての絵柄の代わりになりますが、スキャッターは例外です。
  • 異なるラインで同時に発生する賞金は追加されます。
  • 真ん中のリールに5つのゴールデンシンボルを集めると、5回のフリースピンが有効になります。
  • フリースピン中にゴールデンシンボルを集めると、最大5回まで、合計10回のフリースピンを獲得できます。
  • フリースピンでスキャッターシンボル(リールごとに1つ)が現れると、プライズとマルチプライヤーを獲得できます。
  • ボーナスゲームは勝利ラインに3つ以上のボーナスシンボルを集めると有効になりますが、左から連続するリール上にシンボルがなければなりません。
  • ファンプレーでは、ゲーム履歴は利用できません。
  • 自動プレーメニューを開いてラウンド数を調整し、停止条件を設定します(プレーボタンをクリックして開始し、停止ボタンでいつでも停止できます)。

自動プレー詳細オプション

  • 自動プレーの詳細オプションを設定するには、ゲームパネルの自動プレーシンボルをクリックします。
  • 勝利すると停止。ラウンドで賞金を獲得すると自動プレーを停止します。
  • 次の賞金で停止。自動プレーは、賞金額が一定額を超える、または特定した額になると停止します。
  • キャッシュが次の額に増えると停止。自動プレーは、キャッシュが特定した額に増えると停止します。
  • キャッシュが次の額に減ると停止。自動プレーは、キャッシュが特定した額まで減ると停止します。
  • フィーチャーに入ると停止。自動プレーは、ボーナスラウンドやフリースピンなどのフィーチャーに入ると停止します。
  • 注記: ラウンド中に自動プレー設定を変更すると、すべての設定はゲームラウンドまたはフィーチャーが完了した後に有効になります。
  • 注記: プレー中に切断した場合、ゲームがリロードしたときにすべての自動プレー設定は規定時に戻ります。
  • 注記: 自動プレーの一部のオプションは、司法権が及ぶ地域により必須となります。

ペイントフリースピン

  • 真ん中のリールに5つのゴールデンシンボルを集めた後、5回のフリースピンが有効になります。
  • フリースピンが有効になっているときにさらにゴールデンシンボルを集めると、追加のフリースピンを獲得します。
  • フリースピン中、4つの異なるスキャッターシンボルが有効になります。それぞれ、個別に固定プライズとマルチプライヤーを獲得することができ、それぞれのリールにのみ現れます(詳細は配当表を参照してください)。
  • マルチプライヤーは合計され、フリースピンが終わったときに賞金合計に適用されます。
  • 注記: ボーナスシンボルはフリースピン中には現れません。
  • 注記: 集めたフリースピンシンボルは、賭け金により異なります。特定の賭け金で集めたシンボルは、異なるベットの値では持ち越しされることはありません。その値は記憶され、フリースピンシンボルを集めたときに使用していた賭け金に戻ったときに、その値のフリースピンシンボルの値に戻ります。

ペイントボーナスゲーム

  • ペイントのボーナスゲームは、3つ以上のボーナスシンボルを集めると有効になります。
  • シンボルは左から始まる連続するリールに現れる必要があります。
  • ボーナスシンボルを集めると、ボーナスゲームのマルチプライヤーを増やします(詳細は配当表を参照してください)。
  • ボーナスゲーム中、バブルをクリックしてマルチプライヤープライズを表示させます。
  • ボーナスゲームの終了時に、賞金は、バブルの中に見つけたプライズの合計額で賭け金が乗算され、ラウンドを有効にしたボーナスシンボルマルチプライヤーでもう一度掛けられます。

ゲーム設定オプション

  • ゲーム設定にアクセスするには、左上にあるメニューアイコンをクリックして次に [ギア] アイコンをクリックします。
  • 設定ページ
  • サウンドエフェクト。サウンドエフェクトをオン/オフにします。
  • バックグラウンド音楽。バックグラウンドの音楽をオン/オフにします。
  • スペースを押してスピン。スペースバーを押してスピンを有効/無効にします。
  • ロード時にヘルプ画面を表示。ゲームをロードしたときに、ヘルプ画面をオン/オフにします。
  • ゲーム履歴
  • ゲーム履歴の一覧です。すべてのプレーしたゲーム一覧(ファンモードは除く)で、項目をクリックすると詳細を表示します。
