• Cherry Blast是一款5轉軸、3行的視頻老虎機，擁有免費旋轉和獎勵遊戲。
  • 每次旋轉玩20條固定的贏獎線（每條線1個硬幣），總投注額顯示在投注按鈕上。
  • 要更改投注額，請打開投注面板並拖動轉盤來選擇一項。
  • 在投注設定功能表，以硬幣顯示選項將餘額和賠付值的顯示方式從現金切換為硬幣，投注額始終以現金顯示。硬幣值為：投注額/線數。
  • 使用旋轉按鈕以當前投注額玩遊戲（您還可以上下拖動轉軸以旋轉，但需要先在設定功能表中啟用此功能）。
  • 贏獎組合和賠付在賠付表中有說明。
  • 硬幣線贏獎等於賠付表上顯示的值。
  • 每條活躍的賠付線上僅賠付從左到右排列的獎金最高的組合。
  • Wild符號可替代所有非特殊符號。
  • 不同投注賠付線上同時獲得的獎金將相加。
  • 炸彈符號可能出現在中間轉軸的中部，它會向隨機方向爆炸並顯示出scatter符號、Free Spin符號、Bonus符號或最多5個wild符號。
  • 3個星星符號出現在轉軸上時會啟動9次免費旋轉。
  • 3個scatter符號出現在轉軸上時會贏得80個硬幣。
  • 3個bonus符號出現在轉軸上時會啟動獎勵遊戲。
  • 玩娛樂局時，遊戲歷史記錄不可用。
  • 按住旋轉按鈕來打開自動遊戲功能表，在此調整局數（點擊裡面的開始遊戲按鈕來開始，點擊停止按鈕來隨時停止）。要調整停止條件，請在功能表中打開自動遊戲設定。

自動遊戲進階選項

  • 要設定自動遊戲高級選項，請點擊遊戲左上角的功能表圖示，然後在功能表面板中選擇自動遊戲符號。
  • 在任何贏獎時停止。在一局贏獎時，停止自動遊戲。
  • 贏得指定金額時停止。當獎金額超過或等於指定金額時，停止自動遊戲。
  • 現金增加指定金額時停止。現金增加指定金額時，停止自動遊戲。
  • 現金減少指定金額時停止。現金減少指定金額時，停止自動遊戲。
  • 注：在一局遊戲中更改自動遊戲設定時，所有設定將在遊戲局或特色遊戲完成後生效。
  • 注：如果您在遊戲中斷開連接，所有自動遊戲設定會在重新載入遊戲時恢復為默認。
  • 注：一些自動遊戲選項在某些地區是必選的。

Cherry Blast免費旋轉

  • 在免費旋轉中，2個櫻桃炸彈符號可能出現在轉軸上並顯示超級wild或超級scatter符號。
  • 超級wild和超級scatter符號在免費旋轉中出現，並在2個櫻桃炸彈符號出現在中間三個轉軸上的任意位置後出現。兩個炸彈之間的所有符號（包括炸彈，從轉軸向下走並折回下一個轉軸的頂部，以此類推）將變成超級wild或超級scatter符號，具體取決於出現的符號。如果出現兩個超級wild符號，之間的所有符號將變成wild符號（如果出現兩個超級scatter符號，則都變成scatter符號）。如果每種符號各一個，之間的符號將在wild和scatter符號之間輪流變換。

Cherry Blast Dash獎勵遊戲

  • 在dash獎勵中，玩家可選擇5條路中的1條。
  • 每個黃色格子為賭注的x1翻倍乘數，每個紫色格子擁有不同的獎品。它們在x2翻倍乘數到x60翻倍乘數之間，但紫色格子還可能結束獎勵遊戲。
  • 玩家可決定要冒多大的風險，小路上的紫色格子越多，風險越大，但潛在的贏獎越高。
