• Cosmic CrystalsTM 是一款5转轴、3行的视频老虎机，在旋转未赢奖时有再旋转和粘百搭转轴。
  • 每次旋转玩50条固定的赢奖线（每条线1个硬币），总投注额显示在投注按钮上。
  • 要更改投注额，请打开投注面板并使用箭头调整。
  • 最大注加载允许的最大赌注。
  • 硬币/现金在现金和硬币之间切换余额和赔付值的显示方式。投注额始终以现金显示。硬币值为：投注额除以线数。
  • 使用旋转按钮以当前投注额玩游戏（您还可以按空格键，但需要先在设置菜单中启用）。
  • 赢奖组合和赔付在赔付表中有说明。
  • 硬币线赢奖等于赔付表上显示的值。
  • 每条活跃的赔付线上仅赔付从左到右排列的奖金最高的组合。
  • Wild符号垂直扩展，并仅出现在中间三个转轴上。它们可代替所有其他符号。
  • 不同投注线上同时获得的奖金将相加。
  • 玩娱乐局时，游戏历史记录不可用。
  • 打开自动游戏菜单来调整局数和停止条件（点击开始游戏按钮来开始，点击停止按钮来随时停止自动游戏）。

自动游戏高级选项

  • 要设置自动游戏高级选项，请点击游戏面板中的自动游戏符号。
  • 在任何赢奖时停止。在任何一局赢奖时，停止自动游戏。
  • 赢得指定金额时停止。当奖金额超过或等于指定金额时，停止自动游戏。
  • 现金增加指定金额时停止。现金增加指定金额时，停止自动游戏。
  • 现金减少指定金额时停止。现金减少指定金额时，停止自动游戏。
  • 注：在一局游戏中更改自动游戏设置时，所有更改的设置将在游戏局或特色游戏完成后生效。
  • 注：如果您在游戏中断开连接，所有自动游戏设置会在您重新加载游戏时恢复为默认设置。
  • 注：一些自动游戏选项在某些地区是必选的。

Cosmic CrystalsTM再旋转

  • Cosmic CrystalsTM在每次未赢奖的旋转后会自动免费再旋转。
  • 免费再旋转的次数没有限制，它们会一直自动再旋转，直至出现首个赢奖。
  • 赢奖指游戏在一条赔付线上可赔付的最少硬币数（详见赔付表）。
  • 再旋转以激活再旋转时的相同投注进行。

Cosmic CrystalsTM百搭转轴

  • Cosmic CrystalsTM Wild符号只能出现在转轴2、3和4上。
  • 一个或更多Cosmic CrystalsTM Wild符号出现在转轴2、3或4上时，将会激活Cosmic CrystalsTM百搭转轴特色游戏。
  • 在Cosmic CrystalsTM百搭转轴特色游戏中，Cosmic CrystalsTM Wild符号垂直扩展以覆盖整个转轴。
  • 如果旋转没有赢奖Cosmic CrystalsTM百搭转轴将保持不动，其他转轴再旋转（粘百搭转轴）。
  • 如果再旋转中出现新的Cosmic CrystalsTM Wild符号，它也会垂直扩展。如果仍没有出现赢奖，它会和之前扩展的其他Cosmic CrystalsTM百搭转轴一起保持不动，其他转轴再旋转，直至出现赢奖。
  • Cosmic CrystalsTM百搭转轴特色游戏在出现赢奖后结束。
