• Cosmic CrystalsTM ist ein Video-Spielautomat mit 5 Walzen und 3 Reihen, der Gratis-Respins und Sticky Wild-Walzen bei Spins ohne Gewinn bietet.
  • Sie spielen mit fixen 50 Gewinnlinien pro Spin (1 Münze pro Linie). Der Gesamteinsatz wird auf dem Einsatz-Button angezeigt.
  • Um den Einsatzbetrag zu ändern, öffnen Sie den Einsatzbereich und passen Sie ihn mithilfe der Pfeile an.
  • HÖCHSTEINSATZ lädt den maximal zulässigen Einsatzbetrag.
  • MÜNZEN/CASH wechselt zwischen der Anzeige des Guthabens und der Auszahlungsbeträge von Cash zu Münzen und wieder zurück. Der Einsatzbetrag wird immer in Cash angezeigt. Der Wert der Münzen ist: Einsatzbetrag dividiert durch die Anzahl der Linien.
  • Bei Betätigen des Spin-Buttons wird das Spiel mit dem aktuellen Einsatzbetrag gespielt (Sie können auch die Leertaste drücken, wenn sie Menü "Einstellungen" aktiviert ist).
  • Gewinnkombinationen und Auszahlungen erfolgen gemäß der AUSZAHLUNGSTABELLE.
  • Ein Münzliniengewinn entspricht dem in der AUSZAHLUNGSTABELLE angezeigten Wert.
  • Es wird nur die höchste Gewinnkombination auf einer aktiven Einsatzlinie von links nach rechts ausgezahlt.
  • Wild-Symbole werden vertikal erweitert und erscheinen nur auf den mittleren drei Walzen. Sie können jedes andere Symbol ersetzen.
  • Gleichzeitige Gewinne auf verschiedenen Einsatzlinien werden addiert.
  • Wenn Sie ZUM SPASS SPIELEN, steht kein Spielverlauf zur Verfügung.
  • Öffnen Sie das Menü AUTOPLAY, um die Anzahl der Runden und Stoppkriterien anzupassen (Sie können jederzeit auf den SPIELEN-Button klicken, um das Autoplay zu starten bzw. auf den STOPP-Button, um es zu stoppen).

Erweiterte Autoplay-Optionen

  • Um erweiterte Autoplay-Optionen festzulegen, klicken Sie im Spielbereich auf das Symbol AUTOPLAY.
  • Stoppen bei beliebigem Gewinn. Stoppt das Autoplay, sobald Sie in einer Runde gewinnen.
  • Stoppen bei Gewinn von. Stoppt das Autoplay, wenn der Betrag, den Sie gewinnen, dem festgelegten Betrag entspricht oder diesen übersteigt.
  • Stoppen bei einer Cash-Erhöhung von. Stoppt das Autoplay bei einer Cash-Erhöhung um den angegebenen Betrag.
  • Stoppen bei einer Cash-Verringerung von. Stoppt das Autoplay bei einer Cash-Verringerung um den angegebenen Betrag.
  • Hinweis: Wenn Sie die Autoplay-Einstellungen während einer Spielrunde ändern, werden alle geänderten Einstellungen erst wirksam, nachdem die Spielrunde oder das Feature beendet ist.
  • Hinweis: Wenn Ihre Verbindung während des Spiels unterbrochen wird, werden alle Autoplay-Einstellungen auf die Standardeinstellungen zurückgesetzt, wenn das Spiel neu geladen wird.
  • Hinweis: Einige Autoplay-Optionen sind in einigen Ländern zwingend vorgeschrieben.

Cosmic CrystalsTM Respin

  • Cosmic CrystalsTM bietet nach jedem Spin ohne Gewinn einen automatischen KOSTENLOSEN Respin.
  • Für die kostenlosen, aufeinanderfolgenden Respins gibt es keine Beschränkungen, sie werden automatisch fortgesetzt, bis es eine Gewinnkombination gibt.
  • Dazu müssen Sie den Mindestbetrag an Münzen, die das Spiel in einer einzelnen Gewinnlinie auszahlen kann, gewinnen (siehe AUSZAHLUNGSTABELLE).
  • Respins werden mit dem gleichen Einsatz gespielt wie die Runde, die den Respin ausgelöst hat.

Cosmic CrystalsTM Wild-Walze

  • Die Cosmic CrystalsTM Wild-Symbole können nur auf den Walzen 2, 3 und 4 erscheinen.
  • Wenn ein oder mehrere Cosmic CrystalsTM Wild-Symbole auf den Walzen 2, 3 oder 4 erscheinen, wird das Cosmic CrystalsTM Wild-Walzen-Feature ausgelöst.
  • Während des Cosmic CrystalsTM Wild-Walzen-Features wird das Cosmic CrystalsTM Wild-Symbol vertikal erweitert und deckt die gesamte Walze ab.
  • Wenn der Spin keinen Gewinn bringt, bleibt die Cosmic CrystalsTM Wild-Walze stehen, während die anderen Walzen erneut gedreht werden (Sticky Wild-Walze).
  • Wenn ein neues Cosmic CrystalsTM Wild-Symbol während eines Respins auftaucht, wird es ebenfalls vertikal erweitert. Gibt es keinen Gewinn, bleibt es zusammen mit anderen zuvor erweiterten Cosmic CrystalsTM Wild-Walzen-Symbolen während eines weiteren Respin stehen, bis es einen Spin mit Gewinn gibt.
  • Das Cosmic CrystalsTM Wild-Walzen-Feature endet, sobald es einen Gewinn gibt.
