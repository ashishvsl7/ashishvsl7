Spielregeln für Paint

  • Paint ist ein Videospielautomat mit 5 Walzen und 3 Reihen, der Freispiele und ein Bonusspiel bietet.
  • Sie spielen mit fixen 20 Linien pro Spin (1 Münze pro Linie). Der Gesamteinsatz wird auf dem Einsatz-Button angezeigt.
  • Um den Einsatzbetrag zu ändern, öffnen Sie den Einsatzbereich und passen ihn mithilfe der Pfeile an.
  • HÖCHSTEINSATZ lädt den maximal zulässigen Einsatzbetrag.
  • MÜNZEN/CASH schaltet zwischen der Anzeige des Guthabens und der Auszahlungsbeträge von CASH in Münzen um. Der Einsatzbetrag wird immer in CASH angezeigt. Der Wert einer Münze ist: Einsatzbetrag/Anzahl der Linien.
  • Bei Betätigen des Spin-Buttons wird das Spiel mit dem aktuellen Einsatzbetrag gespielt (Sie können auch die Leertaste drücken, wenn sie in den Einstellungen aktiviert ist).
  • Gewinnkombinationen und Auszahlungen erfolgen gemäß der AUSZAHLUNGSTABELLE.
  • Ein Münzliniengewinn entspricht dem in der AUSZAHLUNGSTABELLE angezeigten Wert.
  • Es wird nur die höchste Gewinnkombination auf einer aktiven Einsatzlinie von links nach rechts ausgezahlt.
  • Wild-Symbole ersetzen alle Symbole außer Scatter-Symbole.
  • Gleichzeitige Gewinne auf verschiedenen Einsatzlinien werden addiert.
  • 5 Freispiele werden aktiviert, wenn Sie 5 der goldenen Symbole auf der mittleren Walze sammeln.
  • Wenn Sie während der Freispiele ein goldenes Symbol sammeln, wird ein zusätzliches Freispiel (maximal 5 zusätzliche Freispiele bei insgesamt 10 Freispielen) vergeben.
  • In Freispielen gibt es Scatter-Symbole (eines pro Walze), die einen Preis und einen Multiplikator vergeben.
  • Das Bonusspiel wird aktiviert, wenn Sie 3 oder mehr Bonussymbole auf einer Gewinnlinie auf benachbarten Walzen (ab der äußersten linken Walze), sammeln.
  • Wenn Sie ZUM SPASS SPIELEN, steht kein Spielverlauf zur Verfügung.
  • Öffnen Sie das Menü AUTOPLAY, um die Anzahl der Runden und Stoppkriterien anzupassen (Sie können jederzeit auf den SPIELEN-Button klicken, um das Autoplay zu starten bzw. auf den STOPP-Button, um es zu stoppen).

Erweiterte Autoplay-Optionen

  • Um erweiterte Autoplay-Optionen festzulegen, klicken Sie im Spielbereich auf das Symbol AUTOPLAY.
  • Stoppen bei beliebigem Gewinn. Autoplay stoppen, wenn Sie in einer Runde gewinnen.
  • Stoppen bei einem Gewinn von Autoplay stoppen, wenn der Betrag, den Sie gewinnen, dem angegebenen Betrag entspricht bzw. diesen übersteigt.
  • Stoppen bei einer Cash-Erhöhung von Autoplay stoppen bei einer Cash-Erhöhung um den angegebenen Betrag.
  • Stoppen bei einer Cash-Verringerung von Autoplay stoppen bei einer Cash-Verringerung um den angegebenen Betrag.
  • Stoppen, wenn ein beliebiges Features gestartet wird. Autoplay stoppen, wenn ein beliebiges Features gestartet wird, z. B. eine Bonusrunde oder Freispiele.
  • Hinweis: Wenn Sie die Autoplay-Einstellungen während einer Spielrunde ändern, werden alle geänderten Einstellungen erst wirksam, nachdem die Spielrunde oder das Feature beendet ist.
  • Hinweis: Wenn Ihre Verbindung während des Spiels unterbrochen wird, werden alle Autoplay-Einstellungen auf die Standardeinstellungen zurückgesetzt, wenn das Spiel neu geladen wird.
  • Hinweis: Einige Autoplay-Optionen sind in einigen Ländern zwingend vorgeschrieben.

Paint-Freispiele

  • 5 Freispiele werden aktiviert, wenn Sie 5 der goldenen Symbole auf der mittleren Walze sammeln.
  • Wenn Sie während der Freispiele weitere goldene Symbole sammeln, wird ein zusätzliches Freispiel vergeben.
  • Es gibt 4 verschiedene Scatter-Symbole während der Freispiele. Jedes davon vergibt einen individuellen, festgelegten Münzpreis und einen Multiplikator. Sie können nur auf ihrer eigenen Walze erscheinen (nähere Infos finden Sie in der AUSZAHLUNGSTABELLE).
  • Multiplikatoren werden addiert und am Ende der Freispiele auf den Gesamtgewinn angewendet.
  • Hinweis: Während der Freispiele erscheinen keine Bonus-Symbole.
  • Hinweis: Die gesammelten Freispielsymbole hängen vom Einsatz ab, das heißt, Symbole, die mit einem bestimmten Einsatz gesammelt werden, werden nicht übernommen, wenn Sie den Einsatzwert ändern. Sie werden gespeichert und sind vorhanden, wenn Sie wieder zum Einsatz zurückkehren, mit dem Sie die Freispielsymbole gesammelt haben.

Paint-Bonusspiel

  • Das Bonusspiel von Paint wird aktiviert, wenn Sie 3 oder mehr Bonussymbole sammeln.
  • Um zu gewinnen, müssen sich die Symbole auf benachbarten Walzen (ab der äußersten linken Walze) befinden.
  • Die Anzahl der gesammelten Bonussymbole erhöht den Multiplikator für das Bonusspiel (nähere Infos finden Sie in der AUSZAHLUNGSTABELLE).
  • Klicken Sie während des Bonusspiels auf Blasen, um Multiplikatorpreise aufzudecken.
  • Am Ende des Bonusspiels ergibt sich Ihr Gewinn aus dem Einsatz, multipliziert mit der Summe der Preise, die Sie in den Blasen gefunden haben, und noch einmal multipliziert mit dem Bonussymbol, das die Runde aktiviert hat.

Spieleinstellungsoptionen

  • Um die Spieleinstellungen zu öffnen, klicken Sie auf das Menüsymbol in der oberen linken Ecke und dann auf das "Rad"-Symbol.
  • Seite "Einstellungen"
  • Toneffekte. Schaltet die Toneffekte ein oder aus.
  • Hintergrundmusik. Schaltet die Hintergrundmusik ein oder aus.
  • Zum Drehen Leertaste drücken. Aktiviert bzw. deaktiviert die Option, einen Spin durch Drücken der Leertaste zu starten.
  • Beim Laden Hilfebildschirm anzeigen. Schaltet den Hilfebildschirm beim Laden des Spiels ein oder aus.
  • Spielverlauf
  • Spielverlaufsliste. Eine Liste aller gespielten Spiele (im SPASS-Modus nicht verfügbar). Klicken Sie auf eine Zeile, um alle verfügbaren Details anzuzeigen.
