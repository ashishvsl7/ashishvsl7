• Gifts of Ostara是一款5轉軸、3行的視頻老虎機，擁有wild轉軸、免費旋轉、scatter、特殊翻倍乘數和再旋轉符號。
  • 每次旋轉玩20條固定的贏獎線（每條線1個硬幣），總投注額顯示在投注按鈕上。
  • 要更改投注額，請打開投注面板並使用箭頭選擇一項。
  • 最大注載入最大的投注額。
  • 硬幣/現金在現金和硬幣之間切換餘額和賠付值的顯示方式，投注額始終以現金顯示。硬幣值為：投注額/線數。
  • 使用旋轉按鈕以當前投注額玩遊戲（還可以按空格鍵，但需要先在設定中啟用）。
  • 贏獎組合和賠付在賠付表中有說明。
  • 硬幣線贏獎等於賠付表上顯示的值。
  • 每條活躍的賠付線上僅賠付從最左邊到右排列的獎金最高的組合。
  • 金蛋特殊符號會在普通遊戲中冒出來並顯示特殊獎品：scatter、翻倍乘數、wild轉軸、粘wild轉軸、再旋轉和最多5個特殊符號。
  • 顯示出scatter獎時，玩家會獲得30x翻倍乘數。
  • 顯示出遊戲翻倍乘數時，玩家會獲得x2、x4、x6、x8或x10翻倍乘數。
  • Wild轉軸代替所有非特殊符號，可在顯示出wild轉軸符號時贏得。
  • 顯示出粘wild轉軸符號時，玩家會獲得3次免費旋轉。
  • 顯示出一隻銀色再旋轉兔子時，玩家會獲得一次免費的再旋轉。
  • 收集到3個排成一行的相同類型的特殊符號時，將會啟動10次免費旋轉。
  • 玩娛樂局時，遊戲歷史記錄不可用。
  • 打開自動遊戲功能表來調整局數和停止條件（點擊開始遊戲按鈕來開始，點擊停止按鈕來隨時停止）。

自動遊戲高級選項

  • 要設定自動遊戲高級選項，請點擊遊戲面板中的自動遊戲符號。
  • 在任何贏獎時停止。在一局贏獎時，停止自動遊戲。
  • 贏得指定金額時停止。當獎金額超過或等於指定金額時，停止自動遊戲。
  • 現金增加指定金額時停止。現金增加指定金額時，停止自動遊戲。
  • 現金減少指定金額時停止。現金減少指定金額時，停止自動遊戲。
  • 注：在一局遊戲中更改自動遊戲設定時，所有設定將在遊戲局或特色遊戲完成後生效。
  • 注：如果您在遊戲中斷開連接，所有自動遊戲設定會在重新載入遊戲時恢復為默認。
  • 注：一些自動遊戲選項在某些地區是必選的。

金蛋特殊符號

  • 金蛋特殊符號會在普通遊戲中從地上的洞裡冒出來並顯示特殊獎品：scatter、翻倍乘數、wild轉軸、粘wild轉軸、再旋轉和最多5個特殊符號。
  • SCATTER
  • 顯示出一隻金兔子時，玩家會獲得適用於當前賭注的30x翻倍乘數。
  • 翻倍乘數
  • 顯示出遊戲翻倍乘數獎時，玩家會獲得適用於當前賭注的x2、x4、x6、x8或x10翻倍乘數。
  • WILD轉軸
  • 顯示出wild轉軸獎時，一個wild轉軸會出現在轉軸上並代替當前旋轉中所有非特殊的符號。
  • 粘WILD轉軸
  • 顯示出粘wild轉軸獎時，玩家會獲得3次免費旋轉。在免費旋轉中，一個wild轉軸會出現在轉軸上並凍結，代替所有非特殊的符號。
  • 再旋轉
  • 顯示出一隻銀色再旋轉兔子時，玩家會獲得一次免費的再旋轉。
  • 彩色蛋特殊符號
  • 在側計數器中顯示和收集到3個排成一行的相同類型的彩色蛋符號時，將會啟動10次免費旋轉。
  • 觸發免費旋轉的彩色蛋出現在免費旋轉的轉軸上時，便會贏得特殊翻倍乘數。
    • 符號10：獎賞翻倍乘數x1
    • 符號J：獎賞翻倍乘數x2
    • 符號Q：獎賞翻倍乘數x3
    • 符號K：獎賞翻倍乘數x4
    • 符號A：獎賞翻倍乘數x5
  • 遊戲結束時，免費旋轉中收集到的特殊符號總數將乘以特殊符號翻倍乘數以及初始旋轉賭注。
