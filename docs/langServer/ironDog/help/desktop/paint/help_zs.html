Paint游戏规则

  • Paint是一款5转轴、3行的视频老虎机，拥有免费旋转和奖励游戏。
  • 每次旋转玩20条固定的赢奖线（每条线1个硬币），总投注额显示在投注按钮上。
  • 要更改投注额，请打开投注面板并使用箭头调整。
  • 最大注加载最大的投注额。
  • 硬币/现金在现金和硬币之间切换余额和赔付值的显示方式，投注额始终以现金显示。硬币值为：投注额/线数。
  • 使用旋转按钮以当前投注额玩游戏（还可以按空格键，但需要先在设置中启用）。
  • 赢奖组合和赔付在赔付表中有说明。
  • 硬币线赢奖等于赔付表上显示的值。
  • 每条活跃的赔付线上仅赔付从左到右排列的奖金最高的组合。
  • Wild符号可替代除scatter外的所有符号。
  • 不同投注赔付线上同时获得的奖金将相加。
  • 在中间转轴上收集5个金色符号可激活5次免费旋转。
  • 在免费旋转中收集金色符号将赢得额外免费旋转，最多可赢得额外5次免费旋转，总计10次。
  • 免费旋转中有scatter符号（每个转轴上一个）奖赏奖品和翻倍乘数。
  • 在一条赢奖线上收集3个或更多bonus符号可激活奖励游戏，这些符号必须从左边开始在连续转轴上排列。
  • 玩娱乐局时，游戏历史记录不可用。
  • 打开自动游戏菜单来调整局数和停止条件（点击开始游戏按钮来开始，点击停止按钮来随时停止）。

自动游戏高级选项

  • 要设置自动游戏高级选项，请点击游戏面板中的自动游戏符号。
  • 在任何赢奖时停止。在任何一局赢奖时，停止自动游戏。
  • 赢得指定金额时停止。当奖金额超过或等于指定金额时，停止自动游戏。
  • 现金增加指定金额时停止。现金增加指定金额时，停止自动游戏。
  • 现金减少指定金额时停止。现金减少指定金额时，停止自动游戏。
  • 进入任何特色游戏时停止。进入特色游戏（如奖励局或免费旋转）时停止自动游戏。
  • 注：在一局游戏中更改自动游戏设置时，所有设置将在游戏局或特色游戏完成后生效。
  • 注：如果您在游戏中断开连接，所有自动游戏设置会在重新加载游戏时恢复为默认。
  • 注：一些自动游戏选项在某些地区是必选的。

Paint免费旋转

  • 在中间转轴上收集5个金色符号后可激活5次免费旋转。
  • 激活免费旋转后，收集更多金色符号可赢得一次额外的免费旋转。
  • 免费旋转中有4种不同的scatter符号，它们奖赏不同的固定奖品和翻倍乘数，并只能出现在各自的转轴上（查看赔付表了解详细信息）。
  • 翻倍乘数加总并在免费旋转结束时与总奖金相乘。
  • 注：Bonus符号不会在免费旋转中出现。
  • 注：收集到的Free spin符号取决于赌注，也就是说，在一种赌注下收集到的符号不能被带入其他赌注额的游戏中。游戏将记忆当前状态，并会在您换回之前收集free spin符号所用的赌注时显示之前收集的符号。

Paint奖励游戏

  • paint中的奖励游戏可通过收集3个或更多bonus符号来激活。
  • 符号必须从左边的转轴开始在连续转轴上排列才能赢奖。
  • 收集到的bonus符号数可增加奖励游戏的翻倍乘数（查看赔付表了解详细信息）。
  • 在奖励游戏中，点击泡泡来显示翻倍乘数奖品。
  • 奖励游戏结束时，奖金等于投注额乘以泡泡中发现的奖品总额，再乘以激活奖励局的bonus符号翻倍乘数。

  • 要访问游戏设置，请点击左上角的菜单图标，然后点击“齿轮”图标。
  • 设置页面
  • 音效。打开或关闭音效。
  • 背景音乐。打开或关闭背景音乐。
  • 按空格以旋转。启用或禁用按空格以旋转的选项。
  • 加载时显示帮助界面。打开或关闭加载游戏时的帮助界面。
  • 游戏历史记录
  • 游戏历史记录列表。所有已玩游戏局的列表（娱乐模式中不可用），点击一行可查看所有详细信息。
