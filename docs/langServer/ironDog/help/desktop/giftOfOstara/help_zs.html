• Gifts of Ostara是一款5转轴、3行的视频老虎机，拥有wild转轴、免费旋转、scatter、特殊翻倍乘数和再旋转符号。
  • 每次旋转玩20条固定的赢奖线（每条线1个硬币），总投注额显示在投注按钮上。
  • 要更改投注额，请打开投注面板并使用箭头选择一项。
  • 最大注加载最大的投注额。
  • 硬币/现金在现金和硬币之间切换余额和赔付值的显示方式，投注额始终以现金显示。硬币值为：投注额/线数。
  • 使用旋转按钮以当前投注额玩游戏（还可以按空格键，但需要先在设置中启用）。
  • 赢奖组合和赔付在赔付表中有说明。
  • 硬币线赢奖等于赔付表上显示的值。
  • 每条活跃的赔付线上仅赔付从最左边到右排列的奖金最高的组合。
  • 金蛋特殊符号会在普通游戏中冒出来并显示特殊奖品：scatter、翻倍乘数、wild转轴、粘wild转轴、再旋转和最多5个特殊符号。
  • 显示出scatter奖时，玩家会获得30x翻倍乘数。
  • 显示出游戏翻倍乘数时，玩家会获得x2、x4、x6、x8或x10翻倍乘数。
  • Wild转轴代替所有非特殊符号，可在显示出wild转轴符号时赢得。
  • 显示出粘wild转轴符号时，玩家会获得3次免费旋转。
  • 显示出一只银色再旋转兔子时，玩家会获得一次免费的再旋转。
  • 收集到3个排成一行的相同类型的特殊符号时，将会激活10次免费旋转。
  • 玩娱乐局时，游戏历史记录不可用。
  • 打开自动游戏菜单来调整局数和停止条件（点击开始游戏按钮来开始，点击停止按钮来随时停止）。

自动游戏高级选项

  • 要设置自动游戏高级选项，请点击游戏面板中的自动游戏符号。
  • 在任何赢奖时停止。在一局赢奖时，停止自动游戏。
  • 赢得指定金额时停止。当奖金额超过或等于指定金额时，停止自动游戏。
  • 现金增加指定金额时停止。现金增加指定金额时，停止自动游戏。
  • 现金减少指定金额时停止。现金减少指定金额时，停止自动游戏。
  • 注：在一局游戏中更改自动游戏设置时，所有设置将在游戏局或特色游戏完成后生效。
  • 注：如果您在游戏中断开连接，所有自动游戏设置会在重新加载游戏时恢复为默认。
  • 注：一些自动游戏选项在某些地区是必选的。

金蛋特殊符号

  • 金蛋特殊符号会在普通游戏中从地上的洞里冒出来并显示特殊奖品：scatter、翻倍乘数、wild转轴、粘wild转轴、再旋转和最多5个特殊符号。
  • SCATTER
  • 显示出一只金兔子时，玩家会获得适用于当前赌注的30x翻倍乘数。
  • 翻倍乘数
  • 显示出游戏翻倍乘数奖时，玩家会获得适用于当前赌注的x2、x4、x6、x8或x10翻倍乘数。
  • WILD转轴
  • 显示出wild转轴奖时，一个wild转轴会出现在转轴上并代替当前旋转中所有非特殊的符号。
  • 粘WILD转轴
  • 显示出粘wild转轴奖时，玩家会获得3次免费旋转。在免费旋转中，一个wild转轴会出现在转轴上并冻结，代替所有非特殊的符号。
  • 再旋转
  • 显示出一只银色再旋转兔子时，玩家会获得一次免费的再旋转。
  • 彩色蛋特殊符号
  • 在侧计数器中显示和收集到3个排成一行的相同类型的彩色蛋符号时，将会激活10次免费旋转。
  • 触发免费旋转的彩色蛋出现在免费旋转的转轴上时，便会赢得特殊翻倍乘数。
    • 符号10：奖赏翻倍乘数x1
    • 符号J：奖赏翻倍乘数x2
    • 符号Q：奖赏翻倍乘数x3
    • 符号K：奖赏翻倍乘数x4
    • 符号A：奖赏翻倍乘数x5
  • 游戏结束时，免费旋转中收集到的特殊符号总数将乘以特殊符号翻倍乘数以及初始旋转赌注。

  • 游戏历史记录列表。所有已玩游戏局的列表（娱乐模式中不可用），点击一行可查看所有详细信息。
