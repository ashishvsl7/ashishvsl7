• Rainbow Wilds是一款5軸、4行、1024條贏獎線路的視頻老虎機遊戲，遊戲中有星星Wild符號、擴展Wild符號、黏性Wild符號、免費跌落以及帶有遞增的倍增指數的免費旋轉。
  • 您每次旋轉玩的線數為固定的50條，總賭注值顯示在下注按鈕上。
  • 若要變更賭注值，請開啟下注面板，使用方向箭頭選擇一個面值。
  • 最大賭注載入最大賭注值。
  • 硬幣/現金可將顯示的餘額和派彩值從現金切換為硬幣，賭注值總是以「現金」顯示。硬幣值為：賭注值除以線數。
  • 旋轉按鈕將以當前的賭注值進行遊戲。
  • 贏獎組合和派彩基於賠付表和贏獎線路的數量進行計算。
  • 贏獎線路的獎金等於賠付表上顯示的值乘以該符號在每個捲軸上出現的次數
  • Wild符號同時算作所有非wild符號。
  • 連續贏得4次或更多的跌落獎金之後，玩家將贏得免費旋轉。
  • 獎勵的免費旋轉次數基於連續跌落的次數。
  • 常規遊戲期間，星星wild符號和擴展wild符號可能出現在第2個、第3個、第4個和第5個捲軸上。黏性wild符號可能出現在中央捲軸上。
  • 遞增的倍增指數在免費旋轉期間啟動。
  • 進行模擬模式時不提供「遊戲歷史記錄」。
  • 開啟「自動下注」功能表以調整回合數目和停止條件（點選「開始遊戲」按鈕可開始遊戲，然後可隨時點選「停止」按鈕停止）。
  • 派彩限額最高為每個遊戲回合的下注本金的6000倍或者300,000枚硬幣。如果派彩因為任何原因超出該數額，那麼派彩將相應減少至該數額。

常規遊戲中的擴展WILD符號

  • 當擴展Wild符號出現在任何捲軸的頂部位置上時，它將擴展以覆蓋整個捲軸。
  • 捲軸上可以同時出現超過一個擴展Wild符號。
  • 擴展Wild符號替代所有非Wild符號，並且能夠出現在除了第1個捲軸之外的所有捲軸上。

免費旋轉

  • 連續贏得跌落獎金之後，玩家將贏得免費旋轉。
    • 連續贏得4次跌落獎金可獎勵14次免費旋轉，沒有額外的起始倍增指數
    • 連續贏得5次跌落獎金可獎勵16次免費旋轉，沒有額外的起始倍增指數
    • 連續贏得6次跌落獎金可獎勵16次免費旋轉，起始倍增指數為3x
    • 連續贏得7次跌落獎金可獎勵20次免費旋轉，起始倍增指數為3x
    • 連續贏得8次跌落獎金可獎勵20次免費旋轉，起始倍增指數為6x

進階自動下注選項

  • 若要設定進階自動下注選項，請在遊戲面板中點選自動下注符號。
  • 贏得任何獎金時停止。在回合中嬴得任何獎金時停止「自動下注」。
  • 單筆獎金為該數額時停止：贏的金額超過或等於指定金額時，停止「自動下注」。
  • 現金增加該金額時停止：現金增加到您指定的金額時，停止「自動下注」。
  • 現金減少該金額時停止：現金減少到您指定的金額時，停止「自動下注」。
  • 進入任何功能時停止。如果觸發任何遊戲功能，那麼「自動下注」將停止。
  • 注意：在遊戲回合期間變更「自動下注」設定時，所有變更的設定會在遊戲回合或功能完成時生效。
  • 注意：如果您的網路在進行遊戲時中斷連線，所有「自動下注」設定在重新載入遊戲時回復為預設值。
  • 注意：某些自動下注選項是某些司法管轄區規定的必要選項。
