• Cherry Blast是一款5转轴、3行的视频老虎机，拥有免费旋转和奖励游戏。
  • 每次旋转玩20条固定的赢奖线（每条线1个硬币），总投注额显示在投注按钮上。
  • 要更改投注额，请打开投注面板并使用箭头选择一项。
  • 最大注加载最大的投注额。
  • 硬币/现金在现金和硬币之间切换余额和赔付值的显示方式，投注额始终以现金显示。硬币值为：投注额/线数。
  • 使用旋转按钮以当前投注额玩游戏（还可以按空格键，但需要先在设置中启用）。
  • 赢奖组合和赔付在赔付表中有说明。
  • 硬币线赢奖等于赔付表上显示的值。
  • 每条活跃的赔付线上仅赔付从左到右排列的奖金最高的组合。
  • Wild符号可替代所有非特殊符号。
  • 不同投注赔付线上同时获得的奖金将相加。
  • 炸弹符号可能出现在中间转轴的中部，它会向随机方向爆炸并显示出scatter符号、free spin符号、bonus符号或最多5个wild符号。
  • 3个星星符号出现在转轴上时会激活9次免费旋转。
  • 3个scatter符号出现在转轴上时会赢得80个硬币。
  • 3个bonus符号出现在转轴上时会激活奖励游戏。
  • 玩娱乐局时，游戏历史记录不可用。
  • 打开自动游戏菜单来调整局数和停止条件（点击开始游戏按钮来开始，点击停止按钮来随时停止）。

自动游戏高级选项

  • 要设置自动游戏高级选项，请点击游戏面板中的自动游戏符号。
  • 在任何赢奖时停止。在一局赢奖时，停止自动游戏。
  • 赢得指定金额时停止。当奖金额超过或等于指定金额时，停止自动游戏。
  • 现金增加指定金额时停止。现金增加指定金额时，停止自动游戏。
  • 现金减少指定金额时停止。现金减少指定金额时，停止自动游戏。
  • 注：在一局游戏中更改自动游戏设置时，所有设置将在游戏局或特色游戏完成后生效。
  • 注：如果您在游戏中断开连接，所有自动游戏设置会在重新加载游戏时恢复为默认。
  • 注：一些自动游戏选项在某些地区是必选的。

Cherry Blast免费旋转

  • 在免费旋转中，2个樱桃炸弹符号可能出现在转轴上，它们爆炸时会显示超级wild或超级scatter符号。
  • 超级wild和超级scatter符号在免费旋转中出现，并在2个樱桃炸弹符号出现在中间三个转轴上的任意位置后出现。两个炸弹之间的所有符号（包括炸弹，从转轴向下走并折回下一个转轴的顶部，以此类推）将变成超级wild或超级scatter符号，具体取决于出现的符号。如果出现两个超级wild符号，之间的所有符号将变成wild符号（如果出现两个超级scatter符号，则都变成scatter符号）。如果每种符号各一个，之间的符号将在wild和scatter符号之间轮流变换。

Cherry Blast Dash奖励游戏

  • 在dash奖励中，玩家可选择5条路中的1条。
  • 每个黄色格子为赌注的x1翻倍乘数，每个紫色格子拥有不同的奖品。它们在x2翻倍乘数到x60翻倍乘数之间，但紫色格子还可能结束奖励游戏。
  • 玩家可决定要冒多大的风险，小路上的紫色格子越多，风险越大，但潜在的赢奖越高。
