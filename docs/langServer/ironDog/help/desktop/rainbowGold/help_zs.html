• Rainbow Wilds是一款5轴、4行、1024条赢奖线路的视频老虎机，游戏中有星星Wild符号、扩展Wild符号、粘性Wild符号、免费下落以及带有递增倍增指数的免费旋转。
  • 每次旋转玩50条固定的赢奖线，总投注额显示在投注按钮上。
  • 如要更改投注额，请打开投注面板并使用箭头选择一项。
  • 最高投注加载最高的投注额。
  • 硬币/现金在现金和硬币之间切换余额和赔付值的显示方式，投注值始终以现金显示。硬币值为：投注值除以线数。
  • 按下旋转按钮将会使用当前的投注值来进行游戏。
  • 赢奖组合和派彩根据赔付表和赢奖线路的数量而决定。
  • 赢奖线路的奖金等于赔付表上显示的值乘以该符号在每个卷轴上出现的次数。
  • Wild符号同时算作所有非wild符号。
  • 连续赢得4次或更多下落奖金之后，玩家将赢得免费旋转。
  • 连续下落次数决定赢得的免费旋转次数。
  • 星星wild符号和扩展wild符号在常规游戏中可能出现在第2个、第3个、第4个和第5个卷轴上。黏性wild符号可能出现在中央卷轴上。
  • 递增的倍增指数在免费旋转游戏中启用。
  • 娱乐模式游戏期间没有游戏历史功能。
  • 打开自动游戏菜单来调整局数和停止条件（点击开始游戏按钮来开始，然后点击停止按钮来随时停止）。
  • 游戏的派彩上限为每局游戏本金的6000倍或者300,000枚硬币。如果派彩出于任何原因超过该数额，那么派彩将相应减少至该数额。

常规游戏期间的扩展WILD符号

  • 当扩展Wild符号出现在任何卷轴的顶部位置时，它将扩展覆盖整个卷轴。
  • 卷轴上可以同时出现超过一个扩展Wild符号。
  • 扩展Wild符号替代所有非Wild符号，并且能出现在除了第1个卷轴之外的任何卷轴上。

免费旋转

  • 连续赢得下落奖金之后，玩家将赢得免费旋转。
    • 连续赢得4次下落奖金会奖励14次免费旋转，没有额外起始倍增指数
    • 连续赢得5次下落奖金会奖励16次免费旋转，没有额外起始倍增指数
    • 连续赢得6次下落奖金会奖励16次免费旋转，起始倍增指数为3x
    • 连续赢得7次下落奖金会奖励20次免费旋转，起始倍增指数为3x
    • 连续赢得8次下落奖金会奖励20次免费旋转，起始倍增指数为6x

自动游戏高级选项

  • 如要设置自动游戏高级选项，请点击游戏面板中的自动游戏符号。
  • 赢得任何奖金时停止。当您赢得任何一局游戏时，自动游戏将停止。
  • 单次奖金超过指定金额时停止。当奖金额超过或等于指定金额时，自动游戏将停止。
  • 现金增加指定金额时停止。当现金增加指定金额时，自动游戏将停止。
  • 现金减少指定金额时停止。当现金减少指定金额时，自动游戏将停止。
  • 进入任何特色游戏时停止。触发任何特色游戏之后停止自动游戏。
  • 注：在一局游戏中更改自动游戏设置时，所有设置将在游戏局或特色游戏完成后生效。
  • 注：如果您在游戏中断开连接，所有自动游戏设置会在重新加载游戏时恢复为默认。
  • 注：一些自动游戏选项在某些地区是必选的。
