• Blood Queen est une machine à sous vidéo à 5 rouleaux et 3 rangées, avec un rouleau Wild pendant le jeu normal, des Tours Gratuits, un rouleau Wild qui se déplace et des Wild Gluants qui s’étendent pendant les Tours Gratuits.
  • Vous jouez 30 lignes par tour (1 pièce par ligne), et la mise totale est affichée sur le bouton de mise.
  • Pour modifier la mise, ouvrez le tableau de mises et sélectionnez-en une à l’aide des flèches.
  • MISE MAX sélectionne la mise maximale.
  • PIÈCES / ESPÈCES fait basculer l’affichage d’ESPÈCES à PIÈCES, sachant que la valeur de mise s’affiche toujours en ESPÈCES. La valeur d’une pièce est : La valeur de la mise / nombre de lignes.
  • Le bouton Spin lance le jeu à la valeur de mise actuelle (vous pouvez appuyer sur la barre d’espace si cette option est activée dans les paramètres).
  • Les combinaisons gagnantes et les paiements sont effectués en fonction du TABLEAU DES PRIX.
  • Un gain de ligne en pièces est égal à la valeur indiquée sur le TABLEAU DES PRIX.
  • Seule compte la meilleure combinaison gagnante sur une ligne de mise de gauche à droite.
  • Les symboles Wild remplacent tous les symboles ordinaires.
  • Les symboles Free Spin peuvent apparaître dans n’importe quelle position sur les rouleaux.
  • Le nombre de tours gratuits attribué dépend du nombre d’icônes Free Spin affichées sur les rouleaux.
  • Pendant le jeu, des symboles Wild apparaissent en haut d’un rouleau et se répandent jusqu’à le couvrir entièrement.
  • Des symboles Wild Gluants et Wild Expansifs sont activés pendant le jeu Free Spin.
  • L’historique du jeu n’est pas disponible lorsque vous jouez EN MODE FUN.
  • Ouvrez le menu JEU AUTOMATIQUE en maintenant le bouton Spin enfoncé pour régler le nombre de chiffres arrondis (cliquez sur le bouton JOUER à l’intérieur pour lancer le jeu et sur le bouton STOP pour l’arrêter à tout moment). Pour ajuster les critères d’arrêt, ouvrez les paramètres de jeu automatique dans le menu.
  • Le jeu est soumis à une limite de gain représentant la mise multipliée par 4000 (majoration effectuée à chaque partie). Si un gain dépasse ce montant pour quelque raison que ce soit, il sera ramené audit montant.

Options avancées de Jeu Automatique

  • Pour définir les options avancées de jeu automatique, cliquez sur l’icône du menu dans le coin supérieur gauche du jeu, puis sélectionnez le symbole JEU AUTOMATIQUE.
  • Arrêter après le prochain gain. Arrête le Jeu Auto lorsque vous gagnez une partie.
  • Arrêter après un gain unique de. Arrête le Jeu Auto lorsque le montant d’un gain est égal ou supérieur au montant indiqué.
  • Arrêter si les gains augmentent de. Arrête le Jeu Auto si l’argent disponible augmente du montant indiqué.
  • Arrêter si les gains diminuent de. Arrête le Jeu Auto si l’argent disponible diminue du montant indiqué.
  • Arrêter lorsqu’une fonction est activée. Arrête le Jeu Auto si une fonction de jeu est activée.
  • Remarque : Lorsque vous modifiez les paramètres de Jeu Automatique pendant une partie, tous les réglages prendront effet une fois la partie terminée.
  • Remarque : Si vous êtes déconnecté pendant le jeu, tous les paramètres de Jeu Automatique reviendront aux valeurs par défaut lorsque vous rechargerez le jeu.
  • Remarque : Certaines options de jeu automatique sont obligatoires dans certains pays.

LES SYMBOLES ROULEAU WILD pendant le jeu normal

  • Lorsqu’une icône de gargouille apparaît sur n’importe quelle position supérieure d’un rouleau, elle s’étendra jusqu’à le couvrir entièrement.
  • Il peut y avoir plus d’un rouleau Wild en même temps.
  • Les rouleaux Wild complètent et remplacent toutes les icônes sauf les symboles Free Spin.

Free Spin

  • Les symboles Free Spin peuvent apparaître dans n’importe quelle position sur les rouleaux.
  • Le nombre de tours gratuits attribué dépend du nombre de symboles Free Spin qui apparaissent sur les rouleaux.
    • 3 symboles = 20 tours gratuits
    • 4 symboles = 30 tours gratuits
    • 5 symboles = 50 tours gratuits
  • Selon la sélection initiale (Van Hell ou Blood Queen), deux symboles Wild différents peuvent être affichés au cours du jeu Free Spin.
  • L’eau bénite de Van Hell accroît les Wild gluants
    • Le symbole de l’eau bénite apparait sur n’importe quelle position sur les rouleaux 2, 3 et 4 et active les Wild gluants. La première position du Wild gluant détermine la direction de l’expansion. L’Eau Bénite se répandra à chaque tour jusqu’à ce qu’elle recouvre 4 symboles, puis elle disparaîtra le tour suivant. L’animation commence verticalement et se déplace horizontalement vers le centre.
    • Il peut y avoir plus d’une icône Wild gluant en même temps.
    • Tout Wild gluant actif s’arrêtera là où il se trouve s’il est joué pendant le dernier tour gratuit.
    • Les symboles Wild remplacent tous les symboles ordinaires.
  • Blood Queen décale le rouleau Wild
    • Le Wild Blood Potion apparaît en haut des rouleaux 2, 3 et 4 et s’étend jusqu’à les couvrir entièrement. Chaque tour, le rouleau Wild se déplacera vers la gauche jusqu’à ce qu’il atteigne le premier rouleau, où il disparaîtra.
    • Il peut y avoir plus d’un rouleau Wild en même temps.
    • Tout rouleau Wild actif s’arrêtera là où il se trouve s’il est joué pendant le dernier tour gratuit et les Wild ne resteront pas en place après la fin des tours gratuits
    • Les symboles Wild remplacent tous les symboles ordinaires.

Options de réglage du jeu

  • Pour accéder à la configuration du jeu, cliquez sur l’icône du menu dans le coin supérieur gauche.
  • La page sonore
  • Effets sonores. Active ou désactive les effets sonores.
  • Musique de fond. Active ou désactive la musique de fond.
  • Page de configuration
  • Jeu pour gaucher. Échange les places des boutons MISE et JOUER.
  • Écran intro. Active ou désactive l’écran d’introduction.
  • Jeu Automatique
  • Jeu auto Voir les options avancées du Jeu Automatique ci-dessus.
  • Historique du jeu
  • Historique des parties. Liste de toutes les parties jouées (non disponible en mode FUN). Cliquez sur une rangée pour voir tous les détails disponibles.
  • Paramètres de mise
  • Afficher en pièces. Bascule entre les pièces et les espèces. Cela ne concerne que le solde total et les gains, les mises étant toujours affichées en espèces.
  • Page d’accueil. Cliquez sur le bouton Accueil pour revenir au lobby.
